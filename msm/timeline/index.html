Version 3.0

1. Redesign Markup ใหม่
    1.1 ใช้ MVC model เพื่อลดการจัดการ element ต่างๆ
    1.2 ปรับการแสดงผลให้แสดงผลแบบ Responsive ได้มากขึ้น
    1.3 ใช้ CSS3 แทนการใช้ loop เพื่อลดภาระของ Client
    1.4 ปิดการแสดงผลส่วนที่ยังไม่ได้พัฒนาเพื่อลดคำถามจาก User
2. ปรับเปลื่ยนการรับส่งข้อมูลแบบ loop req. มาใช้ Web Socket เพื่อลด Traffic ของ Network
3. แก้ไขบั้กในการแสดง Time Bar ไม่ถูกต้อง
4. เพิ่มการแสดงผล
    4.1 แสดงกำหนดเสร็จในสถานะ Setup
    4.2 แสดงความเร็วที่กำหนดในสถานะ Running
    4.3 แสดง Alert ในแถบขวา หาก Setup เกินเวลา หรือ Speed ต่ำกว่าที่กำหนด
    4.4 แสดงรายละเอียดของงานเมื่อ hover ที่ Time Bar
    4.5 เพิ่มปุ่มสำหรับเรียกหน้าจดใหม่
5. เปลื่ยน Service ใหม่
    5.1 เปลื่ยนมาใช้ Web Service เดิมที่ใช้ Client -> PHP -> TCP Service เป็น Client -> Web Service เพื่อลด Layer การทำงาน
    5.2 ใช้ SignalR จัดการ Web Socket เพื่อลด Traffic ของ Network
    5.3 เปลื่ยนการเก็บข้อมูลใน Service จากเดิมที่ loop เก็บไว้ใน Object เป็นเก็บแบบ Object ที่มีโครงสร้างเดี๋ยวกับฐานข้อมูลเลย เพื่อแก้ไขปัญหาเรื่อง Service ทำงานไม่ทัน
    5.4 เลือกส่งข้อมูลที่จำเป็น ลดการส่งข้อมูลที่ซับซ้อน เพื่อลด Traffic ของ Network และอ่านข้อมูลได้ง่ายขึ้น
6. เพิ่มระบบป้องกันการทำงานผิดพลาด หากมีข้อมูลที่อาจเกิดปัญหากับการคำนวน ระบบจะ Reset ตัวเองอัตโนมัติ หรือทุกๆ 30 นาทีเพื่อ Clear cache
